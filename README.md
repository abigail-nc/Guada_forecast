# Recursos y archivos para la elaboración de boletines meteorológicos para el estado de Jalisco

El presente reporsitorio resguarda los recursos y archivos que fueron utilizados durante la elaboración de boletines meterológicos para el estado de Jalisco, enfocado al área metropolitana del estado. La estructura de este mismo se describe en la siguiente tabla

| DIRECTORIO   | CONTENIDO |
|--------------|------------------------|
| _layouts     | Plantillas para el boletín
| +BIBLIOGRAFIA| Libros, apuntes y páginas consultados para elaborar los boletines
| BOLETINES    | Todos los boletines para Jalisco. Desde abril hasta el 15 de junio. DEntro se encuentra el **Boletín extendido** y el **Boletín para el sitio**
| CAPMA_SENEAM | Cartas sinópticas del servicio CAPMA del SENEAM
| GALILEO      | Figuras de salida del modelo WRF del proyecto Galileo inicializado con datos del GFS
| GOES         | Imágenes consultadas a las 14:00Z para la zona estudio
| OtrosRecursos| Imágenes, mapas y otros recursos consultados e incluídos en los boletines
